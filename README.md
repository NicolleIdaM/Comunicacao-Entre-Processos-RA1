# Comunicacao-Entre-Processos-RA1
Aplicação com backend em C++23/C que gerencie a comunicação entre processos utilizando três mecanismos distintos de IPC: Pipes Anônimos, Sockets (locais) e Memória Compartilhada. A lógica de C++ será visualizada e controlada por uma interface gráfica desenvolvida em Python ou HTML/JavaScript.
